Три задачи объединены в консольное меню с централизованной обработкой ввода.

## Структура проекта

```
src/main/java/org/example/
├── Main.java              # Точка входа, интерактивное меню
├── InputHelper.java       # Утилиты для безопасного чтения ввода
├── PhoneValidator.java    # Задача 1 — валидация телефона
├── TextCleaner.java       # Задача 2 — очистка текста

```

## Задачи

### Задача 1 — Валидация номера телефона

Проверяет, соответствует ли номер международному формату:

| Часть | Правило |
|---|---|
| Знак `+` | Обязателен |
| Код страны | 1–3 цифры |
| Код города/оператора | 1–3 цифры |
| Основной номер | 5–8 цифр |
| Разделители | Пробел, дефис или отсутствуют |

```
Ввод:  +7 999 123-45-67   →  валидный
Ввод:  +1-555-123-4567    →  валидный
Ввод:  +441234567890      →  валидный
Ввод:  89991234567        →  невалидный  (нет +)
```

Используемые API: `Pattern.matches()`

---

### Задача 2 — Очистка текста от букв и пробелов

Удаляет все буквы (в том числе кириллицу) и пробельные символы, оставляя цифры и знаки пунктуации.

```
Ввод:   Мой номер: +7 (999) 123-45-67, индекс: 123456.
Вывод:  +7(999)123-45-67,123456.
```

Регулярное выражение: `[\p{L}\s]`  
`\p{L}` — любая Unicode-буква, `\s` — любой пробельный символ.

Используемые API: `String.replaceAll()`

---

### Задача 3 — Форматирование имени пользователя

Валидирует и форматирует имя по правилам:

- Начинается с буквы `[a-zA-Z]`
- Содержит только буквы, цифры и `_`
- Длина от 3 до 20 символов
- Приводится к нижнему регистру
- Повторяющиеся `__` схлопываются в одно `_`

```
Ввод:  User__NAME123  →  user_name123
Ввод:  123user        →  Ошибка: имя должно начинаться с буквы
Ввод:  a              →  Ошибка: длина от 3 до 20 символов
```

Используемые API: `Pattern`, `Matcher`, `matches()`, `replaceAll()`, `toLowerCase()`
